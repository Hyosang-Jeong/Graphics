There is a ImGui that can control following features.
1. Model control - change center object & Draw normal or face normal
2. Shader control - change current shader. Should click reload shader after changing the shader
3. Camera - change camera's position, yaw and pitch
4. Material - change object's ambient, diffuse, specular, emissive, texture projection mode, texture entity and where to calculate..
5. Global constant -  change atteunation, global ambient, fog color, fog min, fog max.
6. Light - Can chage each light's type, color and number of active lights. And also you can choose scenario and start/pause rotation
